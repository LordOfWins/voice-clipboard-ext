Privacy Policy for Voice Clipboard

Last updated: April 18, 2026

Summary: Voice Clipboard does not collect, store, transmit, or share any personal data. All voice recognition happens locally in your browser via the Web Speech API, and recognized text is inserted directly into the focused input field. No data leaves your device through this extension.

1. Data We Collect

None. Voice Clipboard does not collect any personally identifiable information, browsing history, user content, authentication credentials, location, financial data, health information, or any other user data.

2. Microphone Access

Voice Clipboard requests microphone access solely to enable voice-to-text input. The microphone stream is passed directly to your browser's built-in Web Speech API (window.SpeechRecognition). We do not record, store, or transmit audio.

3. Speech Recognition

Voice Clipboard uses the Web Speech API provided by your browser. On Google Chrome, speech recognition may be processed by Google's servers as part of the browser platform itself. This processing is governed by Google's own policies, not by Voice Clipboard. Voice Clipboard has no access to, and does not log, the audio stream or the recognition results beyond inserting the final text into the text field you have focused.

4. Local Storage

Voice Clipboard does not use chrome.storage, cookies, IndexedDB, or any persistent storage. All runtime state (e.g. selected language, floating button position) exists only in memory and is discarded when the tab is closed.

5. Remote Code

Voice Clipboard does not fetch, execute, or evaluate any remote code. All extension logic is bundled inside the installed package.

6. Third-Party Services

Voice Clipboard does not communicate with any third-party server, analytics service, advertising network, or tracking provider.

7. Permissions Justification

  • <all_urls> content script match: Required so users can dictate text into any website's input field (email clients, search bars, messaging apps, docs, forms, etc.). This is the core functionality of the extension.

8. Children's Privacy

Voice Clipboard is not directed at children under 13. It does not knowingly collect personal information from children.

9. Changes to This Policy

If this policy changes, the updated version will be published at this same URL with a new "Last updated" date. Material changes will also be reflected in the Chrome Web Store listing.

10. Contact
